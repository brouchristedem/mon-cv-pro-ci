# Mon CV Pro CI

Application de création de CV professionnels — Next.js + Firebase.

## Configuration Vercel

Dans les paramètres du projet Vercel (Settings → Environment Variables), ajoutez les
variables listées dans `.env.example` avec les vraies valeurs Firebase.

## Firestore

Les règles de sécurité se trouvent dans `firestore.rules` — à copier dans la console
Firebase (Firestore Database → Règles).
